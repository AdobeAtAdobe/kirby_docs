# Adobe Analytics Field Mapping to ExperienceEvent XDM

Adobe Analytics report suites are published to the Adobe Experience Platform as two separate datasets, a postvalues and midvalues. The postvalues dataset reflects the data as used in Analysis Workspace and the midvalues support using the Query Service's Adobe Defined Functions (ADFs) including custom sessionization and attribution. Both datasets are pre-filtered to exclude the standard bot traffic and rows flagged to as excluded from reporting in Analytics.

| Field Prefix | Description |
| --- | --- |
| m_ | Used to populate fields in the midvalues dataset. These mid values have data processing applied (Analytics VISTA and Processing Rules) but no attribution. |
| post_ | Used to populate the fields in the postvalues dataset. These post values have additional processing applied, including attribution, and include visit based fields. |
